# Personal Projects — Pratik Bhala

A collection of data science, data engineering, and analytics projects spanning energy, agriculture, public health, and entertainment data. Each folder is a self-contained project with its own README, notebook(s), data, and outputs.

📧 pratikbhala1998@gmail.com

| Project | What it does | Key tech |
|---------|--------------|----------|
| [**datacentre-energy-analytics**](./datacentre-energy-analytics/) | MSc dissertation. Scrapes global data-centre counts, engineers multi-source energy data into tidy tables, mines a decade of news sentiment from GDELT, trains a LightGBM surrogate for server power draw, and reconstructs UK data-centre demand. | Python, pandas, DuckDB, LightGBM, GDELT, Folium |
| [**agricultural-sustainability-analysis**](./agricultural-sustainability-analysis/) | Analyses agro-food CO2 emissions and land-use efficiency for top agricultural producers; builds tidy summary tables and a sustainability-prediction model with SHAP explainability. | Python, pandas, scikit-learn, SHAP |
| [**crop-yield-prediction-dnn**](./crop-yield-prediction-dnn/) | Builds a clean modelling table from FAOSTAT data, clusters countries, and trains neural networks to predict crop yields, with a small hyperparameter sweep. | Python, TensorFlow/Keras, scikit-learn |
| [**energy-consumption-trends**](./energy-consumption-trends/) | Explores how energy is generated and consumed across countries, comparing developed vs developing nations using Our World in Data. | Python, pandas, matplotlib |
| [**netflix-azure-data-engineering**](./netflix-azure-data-engineering/) | End-to-end data-engineering pipeline on Azure (Databricks) over the Netflix titles dataset. | Azure, Databricks, PySpark |
| [**protein-expression-prediction**](./protein-expression-prediction/) | Supervised modelling to predict protein expression from train/test feature matrices. | Python, scikit-learn |
| [**gaming-impact-children-behaviour**](./gaming-impact-children-behaviour/) | Cleans a children's daily-activity dataset, engineers gaming/physical-activity features, segments participants with K-Means, and visualises behaviour change across intervention phases. | Python, pandas, scikit-learn |

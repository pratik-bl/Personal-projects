# Crop Yield Prediction with Deep Learning

Single-notebook project that builds a clean modelling table from FAOSTAT data, clusters countries, and trains neural networks to predict crop yields. Includes a small hyperparameter sweep and saves clear artefacts for review.

## What this notebook does
1) **Load raw inputs**
   - `FAOSTAT_data_en_1-8-2025 sample data.csv`
   - `Value_of_Production_E_All_Data example.csv`

2) **Transform and join**
   - Cleans columns and types
   - Merges FAOSTAT production and value-of-production tables
   - Writes intermediates:
     - `merged_faostat_production.csv`
     - `combined_pivoted_data.csv`

3) **Prepare modelling data**
   - Selects features and target
   - Handles missing values and scaling as needed
   - Exports `cleaned_data.csv`

4) **Cluster countries**
   - KMeans on selected indicators
   - Saves `clustered_data.csv` with a `cluster` label

5) **Train models (TensorFlow/Keras)**
   - Global baseline model on all rows
   - Per-cluster models for each segment
   - Tracks Loss and MAE across epochs

6) **Hyperparameter tuning (light)**
   - Small grid over learning rate, batch size, and hidden units
   - Exports tuning CSVs and plots MAE heatmaps
